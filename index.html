These metrics are fundamental to financial analysis. Each one provides different insights: NPV helps with investment decisions, IRR measures profitability, Sharpe Ratio evaluates risk-adjusted returns, and Beta measures market sensitivity.

Financial Formulas

Net Present Value (NPV)

\[ NPV = \sum_{t=0}^{n} \frac{CF_t}{(1+r)^t} - C_0 \]

Where:

  • \(CF_t\) = Cash flow at time \(t\)
  • \(r\) = Discount rate
  • \(C_0\) = Initial investment
NPV is crucial for investment decisions. A positive NPV means the investment is profitable. The discount rate reflects the opportunity cost of capital.

Sharpe Ratio

\[ Sharpe\ Ratio = \frac{R_p - R_f}{\sigma_p} \]

Where:

  • \(R_p\) = Portfolio return
  • \(R_f\) = Risk-free rate
  • \(\sigma_p\) = Portfolio standard deviation
The Sharpe Ratio measures risk-adjusted returns. Higher values indicate better risk-adjusted performance. A ratio above 1 is generally considered good, above 2 is very good, and above 3 is excellent.

Black-Scholes Option Pricing

\[ C = S_0 N(d_1) - K e^{-rT} N(d_2) \]

\[ d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}} \]

\[ d_2 = d_1 - \sigma\sqrt{T} \]

The Black-Scholes model is fundamental to options pricing. It assumes log-normal distribution of stock prices and constant volatility. This model revolutionized quantitative finance.

Python Implementation


import numpy as np
import pandas as pd

def calculate_npv(cash_flows, discount_rate):
    """
    Calculate Net Present Value
    
    Parameters:
    cash_flows: array of cash flows
    discount_rate: discount rate (as decimal)
    
    Returns:
    NPV value
    """
    npv = 0
    for t, cf in enumerate(cash_flows):
        npv += cf / (1 + discount_rate) ** t
    return npv

# Example usage
cash_flows = [-1000, 300, 400, 500, 600]
discount_rate = 0.1
result = calculate_npv(cash_flows, discount_rate)
print(f"NPV: ${result:.2f}")
This Python function implements the NPV calculation we saw earlier. Notice how we iterate through each cash flow and discount it appropriately. This is a fundamental building block for financial analysis.

Portfolio Analysis


import numpy as np
from scipy import stats

def sharpe_ratio(returns, risk_free_rate=0.02):
    """Calculate Sharpe Ratio"""
    excess_returns = returns - risk_free_rate
    return np.mean(excess_returns) / np.std(excess_returns)

def portfolio_beta(portfolio_returns, market_returns):
    """Calculate portfolio beta"""
    covariance = np.cov(portfolio_returns, market_returns)[0][1]
    market_variance = np.var(market_returns)
    return covariance / market_variance

Sharpe Ratio measures risk-adjusted returns

Beta indicates market sensitivity

These functions calculate key portfolio metrics. The Sharpe Ratio helps compare different investment strategies, while Beta helps understand how a portfolio moves relative to the market. Beta of 1 means it moves with the market, greater than 1 means more volatile, less than 1 means less volatile.
